# Cybersecurity Awareness Chatbot

PROG6221 Part 1 - Console Application for cybersecurity education.

## Features
- AI voice greeting on startup
- ASCII art logo
- 8 cybersecurity topics with randomized responses
- Input validation
- Enhanced console UI with colors and typing effects

## How to Run
1. Clone the repository
2. Open in Visual Studio 2022
3. Press F5 to run
